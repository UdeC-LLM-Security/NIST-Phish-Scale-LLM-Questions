# Reformulated Phish Scale Questions
This repository contains the reformulated questions used for the paper "Automating the NIST Phish Scale with Language Models: A Reduced Metric for Threat Analysis and Red Team Operations" presented at ICAART 2026.

## NIST Compliance & Licensing
This project incorporates methodologies created by employees of the National Institute of Standards and Technology (NIST).

* __Notice of Modification:__ The NIST Phish Scale User Guide provided in this repository has been modified from the original NIST version, with the purpose to evaluate its usage by LLMs.
* __Nature of Changes:__ Changed the wording of the Phish Scale questions to be better utilized by LLMs in a multi-class classification setting.
* __Date of Change:__ November 2025.

## NIST Phish Scale User Guide Citation
Dawkins, S., & Jacobs, J. (2023). NIST Phish Scale User Guide. (NIST Technical Note 2276). National Institute of Standards and Technology. https://doi.org/10.6028/NIST.TN.2276
